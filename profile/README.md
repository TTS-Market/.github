# 🃏 TTS Market

> **The Trading Card Game marketplace platform built for serious sellers.**

TTS Market is a SaaS platform that empowers trading card game (TCG) retailers and resellers with automated inventory management, real-time market data aggregation, and seamless storefront integrations — so you can focus on growing your business, not managing spreadsheets.

---

## 🚀 What We're Building

We're developing a full-stack commerce platform tailored specifically to the TCG industry. Our services handle everything from product catalog enrichment and market price tracking to multi-channel inventory synchronization and operator tooling.

| Layer | Description |
|---|---|
| **Customer Portal** | Self-service dashboard for tenants to manage inventory, pricing rules, and storefront integrations |
| **Operator Admin** | Internal control plane for platform health monitoring, global configuration, and tenant management |
| **Data Pipeline** | Automated collection and normalization of TCG market data from third-party sources |
| **Inventory Sync** | Real-time synchronization engine that keeps storefronts up to date as inventory and prices shift |
| **Infrastructure** | Cloud-native deployment on Google Cloud Platform with full Infrastructure-as-Code |

---

## 🧩 Technology

Our stack is built for reliability, scalability, and developer velocity:

- **Backend:** Go, Python
- **Frontend:** TypeScript / Next.js
- **Cloud:** Google Cloud Platform (Cloud Run, Cloud Functions, Cloud SQL)
- **Infrastructure:** Terraform
- **CI/CD:** GitHub Actions
- **Storefront Integration:** Shopify
